# Lead Finder Application – IMEX Cargo

This repository documents the planning, delivery, and governance of the Lead Finder Application developed for IMEX Cargo.

## Project Overview
The Lead Finder Application was designed to improve lead generation efficiency by enabling advanced search, filtering, reporting, and CRM integration using the Zoho Creator platform.

## Project Management Focus
- Project scope definition and acceptance criteria
- Stakeholder analysis and communication management
- Schedule and delivery management
- Change control and risk management
- User Acceptance Testing (UAT)
- Delivery governance and sponsor sign-off

## Tools & Platforms
- Zoho Creator
- Zoho CRM
- Project planning and governance documentation

## Role
IT Project Manager – responsible for planning, coordination, stakeholder management, governance, and delivery oversight.
